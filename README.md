🌱 Soil Moisture Monitoring System using ESP8266

## 📌 Project Overview
The Soil Moisture Monitoring System is an IoT project built using the ESP8266 (NodeMCU) and a Soil Moisture Sensor. The system continuously monitors the moisture level of the soil and hosts a web page where users can view real-time soil moisture data. This project is ideal for smart farming, home gardening, and plant monitoring applications.

## 🎯 Objective
Monitor soil moisture in real time.
Display soil moisture data on a web page.
Help users identify when plants need watering.
Learn IoT concepts using ESP8266 and Web Server.

## 🛠 Components Required
ESP8266 NodeMCU
Soil Moisture Sensor

## 🔌 Circuit Connections
Soil Moisture Sensor	ESP8266
VCC	3.3V
GND	GND
AO	A0

## ⚙️ Working Principle
The Soil Moisture Sensor measures the moisture level in the soil.
ESP8266 reads the analog value from the sensor.
The sensor data is processed and converted into a moisture percentage.
A web server running on the ESP8266 displays the moisture level.
Users can access the web page through any device connected to the same Wi-Fi network.

## 🌐 Web Interface Features
Real-time Soil Moisture Value
Moisture Percentage
Soil Status (Dry / Moist / Wet)
Responsive HTML & CSS Design
Auto Refresh Data

## ▶️ How to Run
Connect the Soil Moisture Sensor to the ESP8266.
Open the project in the Arduino IDE.
Install the ESP8266 Board Package if required.
Enter your Wi-Fi SSID and Password in the code.
Upload the code to the ESP8266.
Open the Serial Monitor.
Copy the IP Address displayed.
Open the IP address in your browser.
Monitor the soil moisture through the web page.

## 📊 Sample Output
Moisture (%)	Soil Condition
80–100%	🌊 Wet
40–79%	🌿 Moist
0–39%	🌵 Dry
